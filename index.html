Comandos GIT 

        
            new-item index.html                 <-- Inicializar um arquivo
            git init                            <-- Criar um repositorio
            git add .                           <-- adicionar todos os arquivos
            git add index.html                  <-- adicionar arquivo especifico
            git status                          <-- Para visualizar tudo sobre o repositorio
            git commit -m "Primeiro commit"     <-- commit as mudanças do repositorio
            git log                             <-- explorar o histórico de um repositório
            git remote add origin + link perfil <-- O comando git remote permite criar, ver e excluir conexões com outros repositórios
            git branch -M main                  <-- permite criar, listar, renomear e excluir ramificações
            git push -u origin main             <-- é usado para enviar o conteúdo do repositório local para um repositório remoto OBS: aonde esta origin tem que ta o nome que você colocar.
